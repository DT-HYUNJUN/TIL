# DNS

## 도메인

도메인은 인터넷에서 특정 웹사이트나 서비스를 식별하는 주소를 말한다.

이 주소는 고유한 식별자로 사용되며, 사용자가 해당 웹 사이트에 접속하거나 이메일 주소를 생성하는 데 사용될 수 있다.

일반적으로 도메인은 URL의 일부로 사용되며, URL은 웹 페이지나 리소스의 위치를 나타낸다.

`www.google.com`
- www : 서브 도메인
- google : 도메인 이름
- com : 도메인 확장자

## 도메인과 URL

[URL](../Web/uri_url_urn.md)

도메인은 웹 사이트의 주소를 나타내는 개념인 반면에

URL은 웹 페이지나 리소스의 위치를 나타내는 구체적인 주소다.

URL은 도메인을 포함하고, 도메인은 웹 사이트의 주소를 대표하는 개념이다.

## DNS (Domain Name System) 서버

DNS 서버는 인터넷 상에서 도메인 이름을 IP 주소로 변환하거나, IP 주소를 도메인 이름으로 변환하는 역할을 수행하는 시스템이다.

사용자가 이해하기 쉬운 도메인 이름을 컴퓨터가 이해하는 IP 주소로 맵핑하여 웹 브라우징과 인터넷 서비스 이용을 간편하게 만들어 준다.

예를 들어 사용자가 웹 브라우저에서 `www.google.com`을 입력하면 DNS 서버는 이 도메인 이름에 해당하는 IP 주소인 `142.250.76.132`로 해석한다.
